React Native

This is an app I'm developing using React Native. It's purpose is to provide an intuitive, minimalist solution for tracking combat initiative in a TTRPG such as Dungeons and Dragons. This app uses the The D&D 5e API to give users access to open-source D&D content in their battles! It's still a work-in-progress. Eventually I would like to implement a feature which allows the importing of custom creatures in JSON format. You can view the code and watch for updates on github!

    try {
      let response = await fetch(url);
      let json = await response.json();

One of the more exciting parts of the app, this is where I load the entire monster list from the D&D 5e API and then store it in my application state by firing off a redux action. I do this using the useEffect() hook to load the data right when the view loads. This allows me to use ES6 arrow functions in the app and it's an easy alternative to using ComponentDidMount.
